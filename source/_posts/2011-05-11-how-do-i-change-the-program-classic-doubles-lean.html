In order to change the program please do the following:

Go to the main screen (The one with the list of days)
Then press the menu key on your phone
From the menu you can select Preferences
There you can select the program from the list.

Starting v 4.0 we added 'program manager' to the main menu, it will show you the list of programs. Long-tap on the one you want and set as active. Hit back, you'll see the change in the days
