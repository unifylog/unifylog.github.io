What Is the Account/Internet Permission Used For?

Jul 21st, 2011

The app requires permissions to your google account so you don’t have to manually sign in whenever you want to do a back up to/restore from the cloud. Please note that all we do is ask google for a token to see if your account is valid so we can store your data securely and nobody else has access to it. We however, don’t get access to your password or personal information.

The internet permission is also used to back up/restore from the cloud, as well as for validating the license and gathering anonymous information on the performance of the app. Rest assured that we don’t share any data with third parties.

How Do I Start Over?

May 24th, 2011

We’ve been getting this question since version 1. That’s why we created the Cycles module, here is how it works:

A cycle is nothing but a range of dates for a specific program. for example: P90X Classic from January 1st to March 2nd.

  • It doesn’t have to cover the whole program
  • You can set the start and end dates that you want.
  • When you set a cycle as active, your records will be filtered to display only the ones within that date range.

Having said that, if you want to “start over”, all you need to do is create a new cycle, set the starting date as today and make it active, that will only show what you do from now on. So,  if you want to go back and see what you did before, you can change the dates in the cycle or you can mark it as inactive in the edit screen.

How Do I Change the Program (Classic, Doubles, Lean)

May 11th, 2011

Go to the main screen (The one with the list of days)
Then press the menu key on your phone
From the menu you can select Preferences
There you can select the program from the list.
